# Proyecto Integrador IA

## 📖 Descripción
Este proyecto conecta con la API de OpenAI para responder preguntas en lenguaje natural y devolver resultados en formato JSON estructurado. Además, registra métricas de uso como tokens, latencia y costo estimado.

## 📂 Estructura del proyecto
- `src/` → Script principal (`run_query.py`)
- `prompts/` → Ejemplos de prompting (`main_prompt.txt`)
- `reports/` → Reporte técnico (`PI_report_en.md`)
- `test/` → Pruebas automatizadas (`test_core.py`)
- `metrics/` → Métricas guardadas en JSON
- `.gitignore` → Archivos ignorados (incluye `.env`)
